# Frontend Development Skill

前端开发规范和最佳实践。

## React 规范
- 函数组件 + Hooks（优先）
- TypeScript 严格模式
- CSS-in-JS 或 Tailwind CSS

## Vue 3 规范
- Composition API
- `<script setup>` 语法
- Pinia 状态管理

## Nuxt 3 规范
- 自动导入
- 文件路由
- Nitro 服务器
